# Personalização de Joias (Gravações)
**COMO** Cliente,
**QUERO** inserir nomes ou datas para gravação na peça
**PARA** tornar a joia exclusiva e personalizada para um presente ou recordação.

###*Critérios de Aceitação:*

*Deve existir um campo de texto na página do produto para inserção da frase de gravação.
*O sistema deve limitar o número de caracteres conforme o espaço disponível em cada modelo de peça.
*O usuário deve poder escolher entre pelo menos dois estilos de fonte (ex: Cursiva ou Bastão).

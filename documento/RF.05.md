#Gestão de Inventário e Status de Produção
**COMO** Administrador (Joalheiro),
**QUERO** atualizar o status do pedido (Ex: "Em Produção", "Embalado", "Enviado")
**PARA** que o cliente acompanhe a evolução da sua joia artesanal.

**Critérios de Aceitação:**

*O sistema deve permitir a alteração manual do status de cada item do pedido.
*A cada mudança de status, o sistema deve disparar uma notificação automática (E-mail/WhatsApp) para o cliente.
*O sistema deve permitir o registro do código de rastreamento do seguro de carga assim que postado.
